# EnglishCard Bot 🇬🇧

**EnglishCard** — это Telegram-бот для интерактивного изучения английских слов по методу флеш-карточек. Проект разработан в рамках курсовой работы по курсу «Базы данных» (Нетология).

## ⚡️ Проверка работы (Live Demo)
Бот запущен на сервере и доступен для тестирования в режиме реального времени:
👉 <a href="https://t.me/DiplomEnglishLearnBot" target="_blank">EnglishCardBot</a>

*Примечание: Бот работает на домашнем сервере (Synology NAS), возможны кратковременные перебои при обновлении соединения.*

## 📌 Описание проекта
Бот помогает пользователям расширять словарный запас, предлагая угадывать перевод слов. Главная особенность программы — индивидуальный подход:
* Каждый пользователь может формировать **свой собственный словарь**, добавляя новые слова.
* Изученные слова можно удалять из личного списка.
* Данные пользователей изолированы — новые слова, добавленные одним человеком, не появляются у других.

## 🛠 Технологический стек
* **Язык:** Python 3
* **БД:** PostgreSQL
* **ORM:** SQLAlchemy
* **Telegram API:** pyTelegramBotAPI (TeleBot)
* **Конфигурация:** python-dotenv

## 🚀 Запуск проекта

1. Склонируйте репозиторий.
2. Создайте виртуальное окружение и установите зависимости:
   ```bash
   pip install -r requirements.txt
3. Создайте базу данных PostgreSQL.
4. Создайте файл .env в корне проекта и заполните его по шаблону:
    ```bash
    DB_USER=ваш_пользователь
    DB_PASSWORD=ваш_пароль
    DB_NAME=имя_вашей_базы
    TG_TOKEN=токен_вашего_бота
5. Запустите бота:
    ```bash
   python main.py

При первом запуске скрипт автоматически создаст все необходимые таблицы в базе данных.
